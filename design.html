COLOR THEME

The website adopted a black and white theme. The background color is white. Text falls into three level of grey according to the importance of the information. Detailed anylise is in the following part of this section.

This arrangement works good for the website for the main objective of the website is to display images. A simple and colorless theme serves best to emphasize the details of the image.

#222222

Most important infromation including headlines and activated tabs has this most dark color.

#353535

Detailed infromation in paragraphs has this color to differenciate from headlines.

#777777

Infromation in the background, mainly not active tabs and horizontal lines between sections, has this light grey color.

#FED916

This light yellow color only appered on the logo of the subcompany. The logo is not ajusted to be black and white. This indicates that it is a seperate company.

MAP

The only picture that is not a product of the company is a map. The map is redesigned with the theme color.

Font Choice

The website uses only two fonts, both comercial fonts. This helps to deliver the subtile feeling of delicacy for you do not see these fonts very often.

Bebas Neue

This is the font used in headlines and buttons.

Proxima Nova

This is the font used in all the other places on the website

Layout

The page anylised here is the Project page, which contains the core content of the websit.

The navigation bar is fixed on the top of the page. The content page ocupies the rest of the page. It's worth noticing that there is no space between pictures. This is a good choice because it gives the rendering a strong sense of reality. If there were space between images, the 3D feeling would be lost.

The highlight of the layout is its ability to ajust beautifully to all range of devices and screen sizes.

Layout at Full Width

When the website is displayed on a big screen, it displays the images in three columns. The logo is floated to the right and is in fixed size. The navigation is floated to the left. It may not be quite ovbious here, but the navigation buttons are programmed to only ocupy the left half of the page. This would be clearer when we look at the behavious of the website when the width shrinks

Layout at Medium Width

When the width is not enough for the images to mantain a reasonale size, the website ajust to display 2 columns. The navigation buttons would also be folded into 2 rows. It is quite clear now, that the folding point is decided by the center line of the page.

Layout at Compact Width

When the webpage is displayed on a phone or ajusted to be really narrow. It put all the images in one column. The navigation buttons are folded into three rows. The center line is still not crossed.

Layout at Minimum Width

At a minimum width when it cannot diaplay all its navigation buttons in three lines, it changes the navigation intirely. The logo is put in the center, with a drop down menu on top of it.
