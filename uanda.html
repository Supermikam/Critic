Accessibility

The major problem of the website is lacking alternetives for images. In the project page, all the project preview images have no alternatives. Arguably, this is not a big problem for this particular website, for the image itself is the content and the only content that matters here. Even if all the images are provided with alternatives, it would not really help users using screen readers.

Another problem of the website is misusage of h tags. The animation page started with h3 tag, a proper choice would be h2 tags since the page title is omitted and the h3 tags are used for the title of the animations. And the Marketing page used h2 tag for the title, which should be h1.
